## Getting started

- Basic commands:

```shell
docker version # check docker version on client and server
docker info    # check docker information
docker         # list all commands for docker
```

- Common commands:

```shell
docker-compose
docker image
docker container
docker swarm
docker network
docker service
docker stack
```

- File structrues:

  1. Dockerfile
  2. docker-compose.yml
  3. docker-stack.yml
